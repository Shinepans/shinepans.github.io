###作用
- 提高了代码的可维护性
- 避免函数名和变量冲突

###举例
- 模块
```
'use strict';
var s='hello';
function greet(name)
{
	console.log(s+','+name+'!');
}
module.exports=greet;
```
- 主函数

```
'use strict';
var greet=require('./fun_hello');
var s='Shine';
greet(s);
```
- 调用
	到对应目录
	node main.js 即可对fun_hello进行调用
- 结果
	hello,Shine!
###引入模块
- require
模块名字是比如上面的fun_hello.js,那么模块名字即fun_hello
